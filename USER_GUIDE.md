# BarCraft 用户使用说明

BarCraft 是一个可以安装到手机主屏幕的鸡尾酒灵感 App。它包含经典鸡尾酒图鉴、今日推荐、最近查看、库存材料、原创调酒、学习路径、收藏和品鉴笔记。

## 打开方式

线上地址：

```text
https://yves12138.github.io/barcraft/
```

如果已经添加到手机主屏幕，可以直接点击手机桌面上的 `BarCraft` 图标打开。

## 添加到手机主屏幕

### iPhone / iPad

1. 用 Safari 打开 `https://yves12138.github.io/barcraft/`。
2. 点击底部分享按钮。
3. 选择“添加到主屏幕”。
4. 名称保持 `BarCraft`，点击添加。

添加后，BarCraft 会像普通 App 一样从手机桌面打开。

### Android

1. 用 Chrome 打开 `https://yves12138.github.io/barcraft/`。
2. 点击浏览器菜单。
3. 选择“安装应用”或“添加到主屏幕”。
4. 按提示确认。

## 页面说明

### 首页

首页会根据时间和天气给出一杯今日推荐。推荐卡片包含：

- 推荐酒款的中英双语名称。
- 推荐理由。
- 鸡尾酒图片。
- 查看完整配方入口。
- 家中库存推荐、随机探索、最近查看、今日练习和最近复盘。

如果天气读取不准，可以手动选择晴朗、炎热、下雨、寒冷或阴天。

### 图鉴

图鉴用于查看单杯鸡尾酒的完整信息，包括：

- 成品图。
- 配方。
- 制作方法。
- 风味口感。
- 库存匹配状态。
- 替代建议。
- 练习重点。
- 历史笔记。

可以从首页、酒单、收藏、故事、学习和笔记跳转到对应酒款。

### 酒单

酒单收录 102 款经典鸡尾酒。

- “全部”按英文名称首字母分组。
- 也可以按六大基酒筛选：朗姆、威士忌、伏特加、金酒、白兰地、龙舌兰。
- 酒款卡片会显示是否可立即制作，或者还差几样材料。

### 收藏

在图鉴中点亮收藏按钮后，酒款会出现在“收藏 / 我的酒单”页面。

收藏会保存在当前设备和当前浏览器中。换设备、换浏览器或清理浏览器数据后，收藏不会自动同步。

### 故事

故事馆展示当前酒款的起源和近代发展。历史说法存在争议时，会用“常见说法”“资料不一”等方式表达，不把传说写成确定事实。

### 调酒台

调酒台用于把“家里有什么”变成“现在可以做什么”。

你可以：

- 勾选家中已有材料。
- 使用“库存一键推荐”直接挑出当前最适合查看的一杯。
- 查看可立即制作、差 1 样、差 2 样的经典酒。
- 使用常用材料快捷勾选。
- 生成原创鸡尾酒草案。
- 保存原创配方。

调酒台中的勾选状态会自动保存。

### 学习

学习页面提供 7 天入门路径：

1. 认识基酒
2. 酸甜结构
3. 摇和与稀释
4. 第一次搅拌
5. 冰块与杯型
6. 长饮与气泡
7. 创作原创配方

每一天都可以查看练习酒款，也可以跳到笔记页写练习记录。

### 笔记

笔记用于保存品鉴日志和微调记录。可以记录：

- 对应酒款。
- 练习主题。
- 评分。
- 口感描述。
- 本次微调。

笔记会保存在当前设备和当前浏览器中。

## 数据备份

BarCraft 没有账号系统，所有个人数据都保存在当前浏览器本地。

会被本地保存的数据包括：

- 收藏。
- 最近查看。
- 家中库存。
- 原创配方。
- 学习进度。
- 品鉴笔记。

建议定期在“笔记”页面使用“导出备份”，保存一份 JSON 文件。

需要恢复时，在同一页面使用“导入备份”。导入会覆盖当前浏览器里的收藏、最近查看、库存、笔记、原创配方和学习进度。

## 离线使用

BarCraft 支持 PWA 缓存。添加到手机主屏幕后，核心页面资源会被缓存。

注意：

- 第一次打开需要联网。
- 图片和数据缓存以后，离线状态下通常可以打开主界面。
- 如果刚发布新版，手机可能需要重新打开或等待一会儿才会更新到最新版本。

## 常见问题

### 手机上还是旧版本怎么办？

可以尝试：

1. 关闭 BarCraft 后重新打开。
2. 用浏览器打开线上地址并刷新。
3. 等待几分钟，让 GitHub Pages 和手机缓存更新。
4. 必要时删除主屏幕图标，重新添加。

### 为什么本地预览的数据和手机里的数据不一样？

浏览器会按域名隔离本地数据。以下地址的数据互不相通：

```text
localhost
127.0.0.1
yves12138.github.io
```

如果要迁移数据，请在旧地址导出备份，再在新地址导入备份。

### 换手机后数据会自动同步吗？

不会。当前版本没有账号和云同步。换手机前请先导出备份文件。

### 清理浏览器数据会影响 BarCraft 吗？

会。收藏、最近查看、库存、笔记和原创配方都保存在浏览器本地数据中。清理网站数据后，这些内容可能丢失。

### 可以直接分享给别人用吗？

可以。把线上地址发给对方即可：

```text
https://yves12138.github.io/barcraft/
```

对方添加到手机主屏幕后，会拥有自己设备上的独立本地数据。
