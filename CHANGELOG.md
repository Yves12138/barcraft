# Changelog

BarCraft 的版本记录。格式参考 Keep a Changelog，但按当前项目节奏做了简化。

## [2026-06-21] 发布候选版

### 新增

- 添加 PWA 支持，可从 GitHub Pages 添加到 iPhone / Android 手机主屏幕。
- 添加 `manifest.webmanifest`、`service-worker.js` 和多尺寸 App 图标。
- 新增个人数据备份导出 / 导入功能，覆盖收藏、最近查看、库存、笔记、原创配方和学习进度。
- 新增独立的收藏页“我的酒单”。
- 新增调酒台“库存一键推荐”，可根据当前库存直接跳转到最适合查看的配方。
- 新增首页“随机探索”，可从全部经典、当前可做、差 1 样或收藏中随机抽取一杯。
- 新增首页“最近查看”，自动记录最近浏览的酒款并支持回跳和清空。
- 新增图鉴“分享卡片”，可生成本地预览并下载 PNG。
- 新增最终发布说明 `RELEASE_NOTES.md`。
- 新增用户使用说明 `USER_GUIDE.md`。

### 改进

- 丰富首页推荐文案，今日推荐酒名支持中英双语显示，英文比中文略小。
- 合并库存和原创模拟器为“调酒台”，集中展示库存、可做酒款、差 1 样推荐和原创调酒。
- 优化调酒台工作流和反馈状态。
- 优化手机端底部安全区，滚到底部后内容停在底部控制台上方，不被遮挡。
- 优化图鉴、酒单、故事、学习、笔记等页面在移动端的阅读密度。
- 优化首页手机端推荐图显示，保持图片填满卡片。
- 优化图鉴图片对齐和桌面端排版，减少大片留白。
- 优化前端质感、导航选中状态、按钮悬浮 / 选中状态和标题层级。
- 清理未使用素材，更新项目说明和健康检查记录。

### 修复

- 修复调酒台勾选库存材料导致页面跳动的问题。
- 修复调酒台所有勾选控件点击后导致页面位移的问题。
- 修复调酒台勾选材料时顶部导航和底部控制台误显示按下状态的问题。
- 修复桌面端调酒台库存标签被右侧工作台区域覆盖、点击不命中的问题。
- 修复移动端底部控制台遮挡内容的问题。
- 修复学习页底部安全区问题。
- 加强备份导入校验，避免错误文件覆盖本地数据。

### 验证

- 手机宽度 `390 x 844` 下完成首页、图鉴、酒单、收藏、故事、调酒台、学习、笔记 8 个页面切换检查。
- 每个页面滚到底部后，最后可交互元素均未被底部控制台遮挡。
- 首页“查看完整配方”可跳转到图鉴。
- 酒单筛选可用。
- 调酒台材料勾选滚动位移为 0，控件位置位移为 0，导航无误触状态。
- 当前可见图片无破图。
- 控制台无 error / warn。

### 新增

- 为每款酒配置中文名、英文名、配方、制作方法、风味、故事、库存标签、基酒分类和本地图片。
- 添加首页今日推荐、酒单、故事馆、库存 / 家中已有材料、自定义调酒模拟器和底部控制台导航。

- 故事馆统一为“起源”和“近代发展”两类内容。
- 移除故事馆中机构化露出。
- 调整多批中文酒名译名。
