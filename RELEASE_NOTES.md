## 版本概览

BarCraft 当前版本是一个纯静态 PWA 鸡尾酒 App，可通过 GitHub Pages 发布，也可以添加到 iPhone / Android 手机主屏幕使用。

本版本重点完成：

- 102 款经典鸡尾酒图鉴。
- 首页今日推荐。
- 首页随机探索。
- 首页最近查看。
- 图鉴分享卡片。
- 酒单筛选与首字母分组。
- 故事馆。
- 收藏 / 我的酒单。
- 调酒台：库存、库存一键推荐、可做推荐、原创调酒和原创配方保存。
- 7 天学习路径。
- 品鉴日志。
- 本地 JSON 备份和导入。
- PWA 图标、manifest 和 service worker 缓存。
- 手机端底部安全区、滚动位置和导航状态优化。

## 主要功能

### 今日推荐

首页会根据打开时间和天气状态推荐一杯鸡尾酒，并显示中英双语酒名、推荐理由、图片和完整配方入口。首页还支持按“全部经典、现在能做、差 1 样、我的收藏”随机抽取一杯并跳转到图鉴。

### 经典图鉴

每款酒包含：

- 成品图。
- 配方。
- 制作方法。
- 风味口感。
- 库存匹配。
- 替代建议。
- 练习重点。
- 起源和近代发展。
- 分享卡片预览和 PNG 下载。

### 酒单

酒单共 102 款鸡尾酒。

- 全部页按英文名称首字母排列。
- 基酒页按朗姆、威士忌、伏特加、金酒、白兰地、龙舌兰筛选。
- 多基酒酒款会按实际分类出现。

### 收藏

收藏页独立显示用户收藏的酒款，便于形成自己的私人酒单。

### 调酒台

调酒台支持：

- 家中已有材料勾选。
- 一键查看当前库存最适合推荐的一杯。
- 可立即制作、差 1 样、差 2 样推荐。
- 常用材料快捷勾选。
- 原创鸡尾酒草案生成。
- 原创配方保存。

已针对手机端修复：

- 勾选材料导致页面跳动。
- 勾选材料时顶部导航或底部控制台误显示按下状态。

### 学习与笔记

学习页提供 7 天入门路径。笔记页支持记录品鉴、评分、微调和练习复盘。

### 本地备份

用户可以在笔记页导出 / 导入 JSON 备份。

备份内容包含：

- 收藏。
- 最近查看。
- 库存。
- 原创配方。
- 学习进度。
- 品鉴笔记。

## 已知限制

- 当前没有账号系统。
- 当前没有云同步。
- 收藏、库存、笔记、原创配方和学习进度只保存在当前设备和当前浏览器。
- 清理浏览器网站数据可能导致本地数据丢失。
- `localhost`、`127.0.0.1` 和 GitHub Pages 线上地址的数据彼此隔离。
- 离线能力依赖浏览器缓存，首次打开必须联网。
